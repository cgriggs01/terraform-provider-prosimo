# prosimo_dynamic_risk (Resource)

Use this resource to modify dynamic risk settings.

This resource is usually used along with `terraform-provider-prosimo`.



## Example Usage

```terraform
resource "prosimo_dynamic_risk" "dynamic_risk" {
  threshold  {
    name  =  "alert" 
    enabled  =  true
    value  =  80
  }
    threshold  {
    name  =  "mfa" 
    enabled  =  true
    value  =  80
  }
    threshold  {
    name  =  "lockUser" 
    enabled  =  false
    value  =  90
  }
}
```
